## 비트마스크

### 특정 위치 비트 1 또는 0으로 만들기

```c++

x |= (1 << n); // x의 n번째 비트를 1로 만들기
x &= ~(1 << n); // x의 n번째 비트를 0으로 만들기
x ^= (1 << n);  // x의 n번째 비트가 0이면 1로 1이면 0으로 만들기
(x >> n) & 1; // x의 n번째 비트가 뭔지 확인하기
```
